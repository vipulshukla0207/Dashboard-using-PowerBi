# Dashboard-using-PowerBi
Dashboard using Play Store data

--> Google Play Store Data Analysis Dashboard
Project Overview: 
This project analyzes the Google Play Store dataset to uncover insights about app categories, installs, ratings, and user engagement. The goal is to perform Exploratory Data Analysis (EDA) and build an interactive dashboard to visualize trends and patterns.

--> Objectives:

-Analyze total apps, installs, and ratings

-Compare free vs paid apps

-Identify top-performing categories

-Study user engagement and trends over time

--> Dashboard Highlights 

1- Key Metrics

-Total Apps: 8,197

-Total Installs: 361M+

-Average Rating: 4.26 

-Engagement Ratio: 0.03

2- Visual Insights

-Total Installs by Category

-Categories like Game, Communication, and Tools dominate installs

-High demand for entertainment and utility apps

3- Average Rating by Category

-Most categories maintain ratings between 4.0 – 4.5

-Consistent quality across app categories

-->Rating Trends Over Time

-Ratings show gradual improvement over the years

-Indicates better app quality and user satisfaction

4- Free vs Paid Apps

-~93% apps are Free

-Paid apps represent a very small portion

5-Category-wise Comparison

-Combined analysis of:

-Average Rating

-Total Installs

-Total Reviews
